    **Step 1: Create client/src/lib/types.ts.**
    Mirror the exact same TypeScript interfaces from the server types (FeedbackEntry, FeedbackGroup, AnalyticsData). These are duplicated per RESEARCH.md anti-pattern guidance (no shared package for hackathon).

    **Step 2: Create client/src/lib/api.ts.**
    Fetch wrapper with typed return values:
    - `fetchGroups(): Promise<FeedbackGroup[]>` -- GET /api/groups
    - `fetchGroup(id: string): Promise<FeedbackGroup & { entries: FeedbackEntry[] }>` -- GET /api/groups/:id
    - `fetchAnalytics(): Promise<AnalyticsData>` -- GET /api/analytics
    - Each function: fetch(url), check response.ok, throw on error, return typed JSON
    - No external HTTP library needed -- native fetch is fine

    **Step 3: Create client/src/lib/severity.ts.**
    Severity color mapping from RESEARCH.md code example:
    - severityConfig object with critical/high/medium/low keys
    - Each has: color (Tailwind classes for badge bg/text/border), dot (Tailwind class for dot indicator), chart (OKLCH string for Recharts)
    - Export Severity type

    **Step 4: Create minimal App.tsx with routing shell.**
    Set up BrowserRouter with Routes from `react-router` (NOT react-router-dom):
    - Route "/" redirects to "/dashboard"
    - Route "/dashboard" renders a placeholder DashboardPage component (just a div with "Dashboard" text)
    - Route "/analytics" renders a placeholder AnalyticsPage component (just a div with "Analytics" text)
    - Wrap routes in a min-h-screen bg-background div
    - No Header component yet (that's Plan 03) -- just the routing shell

    **Important:** Import from 'react-router' not 'react-router-dom' (React Router v7 single package).

    **Step 5: Create placeholder screenshot SVGs.**
    Create 6 simple SVG files in client/public/screenshots/ (placeholder-01.svg through placeholder-06.svg). Each should be a 400x300 colored rectangle with the text "Screenshot N" centered. Use different background colors per SVG to visually differentiate them (muted blues, purples, greens).
